JukeBox - Features List

JukeBox was originally created just to add as many presets as possible to Slarmoo's box.

Since then, CoolTasDude has branched out, and started looking for other features to add.

Presets
Of the 1070 presets in JukeBox, 485 are original to JukeBox, and the rest are imported from other BeepMods

JukeBox imports EVERY unique preset from:

Slarmoo's Box, ModBox, Sandbox, BlackBox, TodBox, Midbox, Dogebox2, AwesomeBox, DinoBox, AxoBox, LemmBox, BonBox, BloxBox, Unbox Testing, and shitbox4

JukeBox imports MOST presets from:

AbyssBox, BulbBox, and VoxBox

JukeBox also imports presets from weekly challeges 36 and 68 (also in Unbox), as well as 5 presets from Gelli's "Vanilla Challenge"
Max Values
JukeBox Changes Max values for several song settings

  • Max Tempo: 500 -> 2000
  • Max Bar Count: 1024 -> 4096
  • Octave Shift Range: ±2 -> ±8
    • Imported Features
    Presets are not included in this list

  • AbyssBox song editor layouts
  • AbyssBox file export types (.ogg and .opus)
  • MidBox unison types
  • LemmBox HD wave types
  • LemmBox patch notes format
  • AbyssBox Description Buttons
  • D's Quick Box Main Theme
    • Original Features
    These are features made specifically for JukeBox

  • "Unmodified" presets category, containing an unmodified version of each instrument type (Inspired by BulbBox).
  • "Double Octave" and "Triple Octave" unison types
  • "shorten song url," "add instrument within channel, and "delete instrument within channel" keybinds
  • Custom scale menu notes are flipped upsidedown to match the song editor
  • Custom scale now defaults to blank
